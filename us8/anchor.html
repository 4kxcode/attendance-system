Anchor Tag

Anchor Tag <a> is used to display images in HTML in WWW Webpage.

Click the button Copy Source Code to copy the HTML code and paste in sublime text editor and save as anchor.html and open in browser to render the output.

Syntax

<a href="../anchor.html" target="_blank">anchor.html</a>

<!-- Design a basic webpage structure using HTML in order to demonstrate the anchor tag.-->

<!DOCTYPE html>
<html>
<head>
  <title>Anchor Tag</title>
</head>

<body>
  <h1>Anchor Tag</h1>
  <h2>
    <a href="../anchor.html" target="_blank">anchor.html</a>
  <h2>
</body>

</html>
