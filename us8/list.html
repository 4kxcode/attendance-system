List Tag

List Tag <ul> and <ol> are used to unordered and ordered list in HTML in WWW Webpage.

Click the button Copy Source Code to copy the HTML code and paste in sublime text editor and save as list.html, unorderedlist.html, orderedlist.html, ulollist.html, ulollistall.html and open in browser to render the output.

Syntax

<ul type="disc">
  <li>Sanjana Lama</li>
  <li>Ranjit Kafle</li>
</ul>

<ol type="1">
  <li>Binita Gurung</li>
  <li>Sanjay Adhikari</li>
</ol>
