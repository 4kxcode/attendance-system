Marquee Tag

Marquee Tag <marquee> is used to scroll the text in HTML in WWW Webpage.

Click the button Copy Source Code to copy the HTML code and paste in sublime text editor and save as marquee.html and open in browser to render the output.

Syntax

<marquee width="100%" behavior = "scroll" direction = "right" width="100%" height="22px" style="color: white; background: red;"> United School </marquee>

<!-- Design a basic webpage structure using HTML in order to demonstrate the marquee tag.-->

<!DOCTYPE html>
<html>
<head>
  <title>Marquee Tag</title>
</head>

<body>
  <h1 align="center">Marquee Tag</h1>
  <p align="center">
    <marquee width="100%" behavior = "scroll" direction = "right" width="100%" height="22px" style="color: white; background: red;"> United School </marquee>
  <p>
</body>

</html>

Output


Marquee Tag

United School
