Image Tag

Image Tag <img> is used to display images in HTML in WWW Webpage.

Click the button Copy Source Code to copy the HTML code and paste in sublime text editor and save as img.html and open in browser to render the output.

Syntax

<img src="../images/img/img.png" alt="united school" width="1024px" height="403px">

<!-- Design a basic webpage structure using HTML in order to demonstrate the image tag.-->

<!DOCTYPE html>
<html>
<head>
  <title>Image Tag</title>
</head>

<body>
  <h1 align="center">Image Tag</h1>
  <p align="center">
    <img src="../images/img/img.png" alt="united school" width="1024px" height="403px">
  <p>
</body>

</html>

Image Tag
