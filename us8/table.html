Table Tag

Table Tag <table> is used to create Tables in HTML in WWW Webpage.

Click the button Copy Source Code to copy the HTML code and paste in sublime text editor and save as table.html, table1.html, table2.html, table3.html, table4.html, table5.html, table12345.html
and open in browser to render the output.

Syntax

  <table width="25%" align="center" border="1px">
  <tr>
    <th>Table Head One</th>
    <th>Table Head Two</th>
  </tr>

  <tr>
    <td align="center">Table Data One</td>
    <td align="center">Table Data Two</td>
  </tr>
  </table>

<!-- Design a basic webpage structure using HTML in order to demonstrate the table tag.-->

<!DOCTYPE html>
<html>
<head>
  <title>Table Tag</title>
</head>

<body>
  <h1 align="center">Table Tag</h1>
  <table width="25%" align="center" border="1px">
  <tr>
    <th>Table Head One</th>
    <th>Table Head Two</th>
  </tr>

  <tr>
    <td align="center">Table Data One</td>
    <td align="center">Table Data Two</td>
  </tr>
  </table>
</body>

</html>
